# Enrollments
(*enrollments*)

## Overview

### Available Operations

* [getAll](#getall) - Get all Enrollments
* [create](#create) - Create a new Enrollment
* [get](#get) - Get a specific Enrollment
* [update](#update) - Update an Enrollment
* [delete](#delete) - Delete an Enrollment
* [getForClass](#getforclass) - Get Enrollments for a specific Class in a School

## getAll

To get all Enrollments on the service provider.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  security: {
    clientID: process.env["ONEROSTER_CLIENT_ID"] ?? "",
    clientSecret: process.env["ONEROSTER_CLIENT_SECRET"] ?? "",
  },
});

async function run() {
  const result = await oneRoster.enrollments.getAll({
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    // Handle the page
    console.log(page);
  }
}

run();
```
