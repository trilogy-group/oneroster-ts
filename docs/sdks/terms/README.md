# Terms
(*terms*)

## Overview

### Available Operations

* [list](#list) - Get all Terms
* [get](#get) - Get a specific Term
* [getClasses](#getclasses) - Get Classes for a Term
* [getGradingPeriods](#getgradingperiods) - Get Grading Periods for a Term

## list

To get all Terms on the service provider.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  security: {
    clientID: process.env["ONEROSTER_CLIENT_ID"] ?? "",
    clientSecret: process.env["ONEROSTER_CLIENT_SECRET"] ?? "",
  },
});

async function run() {
  const result = await oneRoster.terms.list({
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    // Handle the page
    console.log(page);
  }
}

run();
```
