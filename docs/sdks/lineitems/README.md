# LineItems
(*lineItems*)

## Overview

### Available Operations

* [list](#list) - Get all Line Items
* [create](#create) - Create a Line Item
* [get](#get) - Get a Line Item
* [update](#update) - Update a Line Item
* [delete](#delete) - Delete a Line Item

## list

Get all of the Line Items on the service provider.

### Example Usage

```typescript
import { OneRoster } from "@superbuilders/oneroster";

const oneRoster = new OneRoster({
  security: {
    clientID: process.env["ONEROSTER_CLIENT_ID"] ?? "",
    clientSecret: process.env["ONEROSTER_CLIENT_SECRET"] ?? "",
  },
});

async function run() {
  const result = await oneRoster.lineItems.list({
    fields: "sourcedId,name",
    filter: "status='active'",
  });

  for await (const page of result) {
    // Handle the page
    console.log(page);
  }
}

run();
```

## create

To create a new Line Item. The responding system must return the set of sourcedIds that have been allocated to the newly created Line Item records.

### Example Usage

```typescript

  security: {
    clientID: process.env["ONEROSTER_CLIENT_ID"] ?? "",
    clientSecret: process.env["ONEROSTER_CLIENT_SECRET"] ?? "",
  },
});

async function run() {
  const result = await oneRoster.lineItems.create({
    lineItem: {
      status: "active",
      title: "<value>",
      assignDate: new Date("2025-07-17T09:24:39.378Z"),
      dueDate: new Date("2024-12-24T04:46:32.360Z"),
      class: {
        sourcedId: "<id>",
      },
      school: {
        sourcedId: "<id>",
      },
      category: {
        sourcedId: "<id>",
      },
    },
  });

  // Handle the result
  console.log(result);
}

run();
```
